# 법률 준수 설계

> 최종 수정: 2026-03-30

---

## 1. 개인정보 보호법 준수

### 1.1 수집·이용 동의 (제15조)

| 항목 | 구현 |
|------|------|
| 기기 동의 | 최초 등록 시 동의 화면 표시 (display.c) |
| 웹 동의 | 회원가입 시 체크박스 + 처리방침 링크 |
| 동의 기록 | users.consent_at에 동의 일시 저장 |
| 동의 거부 처리 | 거부 시 서비스 이용 제한 안내 |

### 1.2 개인정보 처리방침 (제30조)

- `/privacy-policy` 페이지에 전문 게시
- 필수 포함 항목 8가지 모두 포함 (부록 A 참조)
- 변경 시 웹사이트에 7일 전 공지

### 1.3 최소 수집 원칙 (제16조)

- 전화번호: 사용자 식별 (필수)
- 이메일: 계정 만료 사전 안내 (필수)
- 그 외 개인정보 수집 없음

### 1.4 안전성 확보 (제29조)

- 전화번호: bcrypt 해시
- 이메일: AES-256 암호화
- 접근 통제: 세션 기반 인증
- 접속 기록: access_log 테이블 (1년 보관)

### 1.5 파기 (제21조)

| 조건 | 처리 |
|------|------|
| 보유기간 경과 | 마지막 로그인 1년 후 자동 파기 |
| 사전 안내 | 만료 1주일 전 이메일 안내 |
| 탈퇴 요청 | 소프트 삭제 즉시 → 30일 후 하드 삭제 |
| 파기 방법 | 복원 불가능한 방식으로 완전 삭제 |

### 1.6 열람·삭제 요청 (제35·36조)

- `/api/user/profile`: 본인 정보 열람 (마스킹)
- `/api/user/account` DELETE: 회원 탈퇴 요청

## 2. 정보통신망법 준수

### 2.1 접속 기록 보관 (제49조의2)

- access_log 테이블에 최소 1년 보관
- 기록 대상: 로그인/로그아웃/기록조회 등 모든 개인정보 처리 이벤트

### 2.2 기술적 보호조치 (제28조)

- 전화번호 해시 + 이메일 암호화
- SSL/TLS 통신
- Rate Limiting
- SQL Injection / XSS / CSRF 방어

## 3. 전기통신사업법

- 전화번호는 사용자 식별 목적으로만 사용
- 마케팅, 제3자 제공 금지
