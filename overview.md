# C언어 프로젝트

- 주제: 헬스장에 IoT를 도입하여 기구를 효율적으로 활용하도록 하는 웹 기반 프로그램
- 기능:
    - 기구마다 사용 중인지, 사용 중이 아닌지 체크하여 사용 기구 현황을 파악할 수 있는 기능
    - 사용자는 기구를 사용하기 전 기구에 탑재되어 있는 기기에 전화번호를 통해 로그인하고, 로그인 이후 대시보드 화면으로 전환되면 오늘 헬스장에서 기구를 통해 소모한 총 칼로리, 현재 사용 중인 기구를 통해 소모 중인 칼로리, 총 운동 시간, 현 기구 사용 시간 등 사용자가 기구를 사용하고 헬스장을 사용할 때 활용하면 좋은 정보들을 띄워줄 수 있어야해
- 요구사항:
    - c언어를 반드시 사용해야함
    - 대학교 2학년 전공자 4명이 2달 내에 개발을 완료할 수 있어야함
    - 대한민국에서 특허를 등록할 수 있어야함
    - 대한민국 현행법 상 위반되는 부분이 단 하나도 없어야함.
