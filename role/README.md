## 역할 간 의존 관계

```
멤버 A (C 서버 코어)
  │
  ├──▶ 멤버 C (기기): TCP 프로토콜 인터페이스 합의
  ├──▶ 멤버 B (칼로리): 세션 종료 시 칼로리 정산 호출
  └──▶ 멤버 D (웹): Redis 기구 상태 캐시 형식 합의

멤버 B (C 서버 보조)
  │
  ├──▶ 멤버 A: DB CRUD 공유, 워치독↔세션관리자 연계
  ├──▶ 멤버 C: 프로토콜 모듈 공유
  └──▶ 멤버 D: daily_stats ↔ 웹 API 데이터 정합성

멤버 C (기기 + 웹 일부)
  │
  ├──▶ 멤버 A: TCP 통신 테스트 함께 진행
  └──▶ 멤버 D: TypeFilter, EquipmentDetail 등 웹 컴포넌트 공유

멤버 D (웹)
  │
  ├──▶ 멤버 B: Redis 캐시 데이터 형식 합의
  └──▶ 멤버 C: 기구 현황 페이지 협업
```
